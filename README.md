## 🗂️ Proyectos Desarrollados

### 🎵 [Machine Learning No Supervisado - Análisis de Clustering Musical](./ML_NO_SUPERVISADO/)
**Agrupamiento de países por preferencias musicales utilizando algoritmos de clustering**

- **🔧 Técnicas**: K-Means, Clustering Jerárquico, DBSCAN, PCA, t-SNE
- **📊 Dataset**: 8 países, 8 géneros musicales  
- **🎯 Objetivo**: Identificar patrones culturales en consumo musical global
- **📈 Resultados**: 5 clusters óptimos con validación por Silhouette Score
- **💼 Aplicación**: Segmentación de mercados para plataformas de streaming

**Competencias**: Análisis exploratorio, clustering, reducción de dimensionalidad, visualización avanzada

---

### 🧠 [Redes Neuronales - Análisis de Sentimientos IMDB](./RED%20NEURONAL/)
**Clasificación automática de sentimientos en reseñas de películas con RNN**

- **🔧 Técnicas**: RNN, Word Embeddings, NLP, Deep Learning
- **📊 Dataset**: 50,000 reseñas de películas etiquetadas (IMDB)
- **🎯 Objetivo**: Clasificación binaria de sentimientos (positivo/negativo)
- **📈 Resultados**: Modelo RNN con arquitectura optimizada y regularización
- **💼 Aplicación**: Sistemas de recomendación y monitoreo de opiniones

**Competencias**: Deep Learning, TensorFlow/Keras, procesamiento de lenguaje natural, análisis de texto

---

### 📈 [Estadística Inferencial - Teoremas Fundamentales](./ESTADISTICA%20INFERENCIAL/)
**Demostración práctica de Ley de Grandes Números y Teorema del Límite Central**

- **🔧 Técnicas**: Simulación estadística, distribuciones de probabilidad, inferencia
- **📊 Experimentos**: Simulaciones Monte Carlo, muestreo estadístico
- **🎯 Objetivo**: Validar teoremas fundamentales mediante experimentos computacionales
- **📈 Resultados**: Verificación empírica de convergencia y normalidad
- **💼 Aplicación**: Fundamentos para análisis estadístico y validación de modelos

**Competencias**: Estadística inferencial, simulación, visualización de distribuciones, análisis probabilístico

---

### 🔄 [ETL - Extract, Transform, Load](./ETL/)
**Pipeline completo de procesamiento y transformación de datos**

- **🔧 Técnicas**: Ingeniería de datos, limpieza, transformación, automatización
- **📊 Dataset**: Datos empresariales con múltiples fuentes
- **🎯 Objetivo**: Crear pipeline robusto para procesamiento de datos
- **📈 Resultados**: Sistema automatizado de ETL con validaciones
- **💼 Aplicación**: Preparación de datos para análisis y modelado

**Competencias**: Ingeniería de datos, pandas, automatización, calidad de datos

---

### 🤖 [Machine Learning Supervisado - Clasificación](./ML/)
**Modelos predictivos para clasificación y análisis de patrones**

- **🔧 Técnicas**: Algoritmos supervisados, validación cruzada, optimización
- **📊 Datasets**: Múltiples conjuntos de datos empresariales
- **🎯 Objetivo**: Desarrollar modelos predictivos de alta precisión
- **📈 Resultados**: Modelos optimizados con métricas de rendimiento validadas
- **💼 Aplicación**: Predicción de comportamiento y toma de decisiones

**Competencias**: Scikit-learn, validación de modelos, feature engineering, optimización de hiperparámetros
